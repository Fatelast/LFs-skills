# A股盘前短线选股：主线趋势前排 + 5日线承接

## 核心定位

使用本 skill 时，只输出“盘前短线观察池”和“盘中确认条件”，不把观察池写成开盘买入清单。

默认适用范围：
- 市场：A股。
- 周期：1-5 个交易日的短线观察。
- 风格：主线趋势前排、5日线承接、竞价与分时确认、破位失效。
- 目标：判断次日是否值得观察、观察什么方向、哪些股票需要盘中确认。

## 硬性边界

- 不输出“必买”“一定上涨”“开盘直接买入”等确定性结论。
- 不编造实时行情、资金流、龙虎榜、公告、财报、监管信息或社媒热度。
- 没有最新交易日数据时，不给具体股票观察池，只列缺失数据和需要补充的来源。
- 盘前不能把分时承接判定为已确认，只能写“待 9:30-9:45 验证”。
- 不根据用户未提供的资产规模、风险承受能力、持仓成本输出个性化仓位指令。
- 如果用户只问单点问题，只回答该问题的关键结论和少量依据，不主动展开完整报告。

## 数据可信度

每次输出前先标注数据时间和可信度。

数据优先级：
1. A类：交易所公告、上市公司公告、定期报告、监管函、问询函、澄清公告。
2. B类：主流行情终端的指数、涨跌家数、涨跌停、成交额、板块涨幅、个股 OHLCV。
3. C类：龙虎榜、融资融券、机构/游资席位、资金流向、新闻快讯。
4. D类：自媒体观点、社媒热度、论坛情绪、博主复盘。

使用规则：
- A类与其他来源冲突时，以 A类为准。
- B类行情至少要有数据日期；无法确认日期时，可信度降为低。
- C类只能作为参与者结构线索，不能单独作为入选理由。
- D类只能作为情绪参考，不能作为事实依据。
- 数据缺失时必须说明“缺失项”和“结论受限点”。

可信度标签：
- 高：行情、板块、公告/财报、个股关键数据均有明确日期和来源。
- 中：行情与板块完整，但公告/财报/资金参与者线索不完整。
- 低：只有零散信息、用户口述或无法确认日期的数据。

## 工作流

按以下顺序执行，不要跳步：

```text
数据校验 -> 市场环境 -> 主线方向 -> 个股地位 -> 技术位置 -> 公告财报 -> 资金参与者 -> 盘中确认 -> 风险失效
```

### 1. 数据校验

必须先确认：
- 最新交易日日期。
- 指数状态：上证、深成指、创业板、科创50。
- 市场温度：上涨/下跌家数、涨停/跌停数、连板高度、昨日涨停溢价。
- 板块强弱：涨停数量、成交活跃度、连续性、前排反馈。
- 个股数据：收盘价、涨跌幅、成交额、换手率、5日线位置、是否涨停或创新高。
- 风险数据：公告、澄清、减持、监管、财报、业绩预告。

如果无法取得上述关键数据，停止给具体股票池，输出待补数据清单。

### 2. 市场环境过滤

判断市场属于以下哪一类：

| 状态 | 判断线索 | 输出态度 |
|---|---|---|
| 积极观察 | 指数修复或震荡向上、涨停增加、主线延续、昨日强势股有溢价 | 可给 1-3 只观察标的 |
| 谨慎观察 | 指数与个股背离、下跌家数偏多、高位分化、赚钱效应集中 | 只给主线核心，降低结论强度 |
| 防守观察 | 跌停增加、高位补跌、主线前排炸板、指数放量下杀 | 原则上不给新观察池 |
| 不适合新开仓 | 无清晰主线、昨日涨停无溢价、全市场明显退潮 | 只输出“等待市场企稳” |

### 3. 主线方向筛选

只看当前最强的 1-2 条主线，不扩散到后排题材。

优先线索：
- 板块连续 2 个交易日以上活跃。
- 涨停数量、成交额、涨幅榜和人气股集中在同一方向。
- 有龙头、中军、补涨核心的层级结构。
- 前排没有明显负反馈，后排没有批量掉队。
- 有可验证产业逻辑、政策催化、订单/业绩/产品事件。

回避线索：
- 单日消息刺激但无板块效应。
- 后排跟风、低辨识度、无成交承接。
- 主线退潮后的补跌票。
- 纯社媒热度但没有行情与成交验证。

### 4. 个股入池条件

候选股必须同时满足大部分条件：
- 属于当前最强主线或次强主线。
- 是龙头、趋势前排、强势中军、高辨识度人气股或补涨核心。
- 成交活跃，不能是无量小票。
- 仍在 5日线体系内，或回踩 5日线后没有有效跌破。
- 近期趋势没有明显破坏，没有连续高开低走或放量长上影。
- 公告、财报、监管、减持等没有明显负面风险。

默认排除或降级：
- ST、退市风险、重大监管风险个股。
- 成交额过低、流动性不足、明显庄股化个股。
- 高波动 20cm、北交所等特殊交易属性标的，除非用户明确要求纳入。
- 远离 5日线过多、连续加速后风险收益不对称的个股。

### 5. 5日线与技术位置

优先形态：
- 强势上涨后首次回踩 5日线。
- 5日线上方缩量整理。
- 高位横盘后右侧重新企稳。
- 主线分歧日仍抗跌。
- 低开后能快速收回关键均线。

失效形态：
- 跌破 5日线且无法收回。
- 放量长上影或连续冲高回落。
- 高开低走并跌破分时均线。
- 主线退潮时的后排补跌。
- 指数放量下杀且个股承接弱于板块。

## 盘中确认门槛

盘前结论必须写成“如果出现 X，则继续观察；如果出现 Y，则放弃”。

竞价与开盘阶段：
- 9:15-9:20：集合竞价可撤单，异常强弱只作参考。
- 9:20-9:25：不可撤单，竞价结果可信度提高。
- 9:25：看是否高开过多、量能是否异常、同板块前排是否同步。
- 9:30-9:45：验证分时承接，不满足则盘前计划失效。

可继续观察的盘中条件：
- 高开不多，回落不破关键均线或 5日线。
- 低开后快速翻红，量能健康。
- 回踩分时均线后重新放量走强。
- 板块前排稳定，个股没有明显弱于板块。
- 指数没有放量下杀。

立即降级或放弃的条件：
- 高开过多直接冲高兑现。
- 开盘急涨急跌，量价失控。
- 跌破 5日线或分时均线后无法收回。
- 板块前排炸板或同题材后排批量回落。
- 突发公告、澄清、监管、减持、业绩风险改变原计划。

## 公告、财报与情绪检查

盘前必须检查并简述：
- 是否有澄清热门概念公告；澄清可能削弱题材情绪。
- 是否有减持、解禁、监管函、问询函、立案调查、风险提示。
- 是否临近财报、业绩预告、业绩快报或重大事项披露。
- 财报是否存在营收/利润异常、现金流压力、商誉减值、连续亏损。
- 所属板块是否仍是热门主线，还是已经进入退潮或轮动末端。

## 资金参与者结构

只能基于可验证线索判断，不能臆测。

可用线索：
- 龙虎榜席位：游资、机构、量化席位只能在有公开榜单时引用。
- 成交额与换手：判断资金活跃度，但不能直接等同于某类资金买入。
- 分时形态：只描述承接强弱，不推断具体账户类型。
- 融资融券：用于观察杠杆资金变化。
- 机构调研、基金持仓、定期报告：用于辅助判断机构关注度。

输出格式使用“可能参与者结构”，例如：
- 游资线索：强 / 中 / 弱 / 无公开依据。
- 量化线索：强 / 中 / 弱 / 无公开依据。
- 机构线索：强 / 中 / 弱 / 无公开依据。
- 散户情绪：强 / 中 / 弱 / 无公开依据。

## 评分框架

评分只用于排序观察池，不代表买入建议。

| 维度 | 分值 | 说明 |
|---|---:|---|
| 市场环境 | 20 | 是否适合短线新观察 |
| 板块强度 | 20 | 是否为当前最强主线 |
| 个股地位 | 20 | 是否前排、龙头、中军或核心补涨 |
| 技术位置 | 15 | 是否在 5日线体系内，风险收益是否合理 |
| 公告财报风险 | 15 | 是否存在负面公告、财报、监管、减持风险 |
| 数据可信度 | 10 | 数据是否完整、及时、可验证 |

分数解释：
- 80 分以上：重点观察，等待盘中触发条件。
- 65-79 分：备选观察，必须降低结论强度。
- 50-64 分：只看不参与，除非盘中显著超预期。
- 50 分以下：剔除。

分时承接不进入盘前评分，只作为盘中确认项。

## 默认输出契约

默认输出必须简短，优先给关键结论。除非用户明确要求完整报告，不展开所有分析细节。

```md
## 盘前结论

数据时间：YYYY-MM-DD 收盘后 / 盘前
数据可信度：高 / 中 / 低
市场态度：积极观察 / 谨慎观察 / 防守观察 / 不适合新开仓
主线方向：……
观察结论：给 1-3 只观察标的 / 暂不给新观察池

## 观察池

| 优先级 | 股票 | 角色 | 盘前依据 | 盘中触发条件 | 失效条件 | 主要风险 |
|---|---|---|---|---|---|---|

## 执行纪律

- 盘前只定观察池，不开盘无条件参与。
- 9:25 看竞价，9:30-9:45 看承接。
- 高开过多、板块退潮、跌破关键位置时，计划失效。
- 如需完整拆解，可继续追问“展开某只股票/板块/资金结构/公告财报风险”。
```

## 单点问题输出

当用户只问一个具体问题时，只回答该问题。

示例：
- 问“明天是否适合开仓”：只回答市场环境、主线状态、是否适合新观察。
- 问“某股能不能进观察池”：只回答入池结论、关键依据、盘中触发、失效条件。
- 问“竞价怎么看”：只回答 9:25 竞价强弱、同板块联动、是否高开过多。
- 问“尾盘是否保留观察”：只回答收盘结构、5日线状态、板块反馈、次日风险。

## 详细输出触发

只有用户明确要求“完整分析”“展开全部”“给详细报告”“逐项评分”时，才输出：
- 市场环境完整判断。
- 板块主线强弱排序。
- 个股逐项评分。
- 公告财报检查。
- 资金参与者结构。
- 技术形态与 5日线分析。
- 盘中确认计划。
- 风险与失效清单。

## 常见错误

- 把博主复盘结论当成事实来源。
- 用单日涨幅替代主线强度。
- 把后排补涨股误判为前排。
- 盘前直接判断“分时承接强”。
- 忽略澄清公告、监管、减持和财报风险。
- 在市场退潮时仍给出积极观察池。
- 因为股票在 5日线上就忽略板块退潮和高位亏钱效应。

## 默认风险提示

本 skill 只用于 A股短线盘前观察池筛选和交易计划整理，不构成投资建议。任何观察标的都必须结合次日竞价、分时承接、板块强弱、公告财报风险和个人风险承受能力再独立决策。
