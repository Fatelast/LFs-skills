# 股票短线分析 Skill

## 1. Skill 定位

该 Skill 用于对 A 股个股进行短线分析，重点回答：

- 当前能不能追？
- 哪只更值得关注？
- 更合理的观察 / 介入观察位置在哪里？
- 什么情况说明走强？什么情况说明转弱？
- 当前市场情绪、板块强弱、资金结构、财报与公告风险是否支持短线继续博弈？
- 多只股票之间，谁的题材、资金、技术形态、情绪地位和风险收益比更优？

适用场景：

- 用户输入 1～3 只 A 股股票代码或名称，要求短线分析。
- 用户询问“能不能追”“明天怎么看”“下午怎么看”“是否有第二波”。
- 用户提供截图、盘口数据、其他 AI 分析内容，希望进行复核、补充或横向比较。
- 用户希望结合市场情绪、板块热度、龙虎榜、融资融券、财报、公告和技术位置判断短线机会。

默认分析周期：短线 1～5 个交易日；必要时可补充 5～20 个交易日的趋势背景。

本 Skill 输出的是基于公开信息和短线交易逻辑的分析框架，不构成个性化投资建议，不替用户做最终交易决策。

---

## 2. 核心判断顺序

短线分析不要先陷入单个技术指标。优先按下面顺序判断：

```text
数据可信度
→ 市场情绪
→ 板块强弱与热门程度
→ 个股板块地位
→ 参与者结构 / 筹码成分
→ 财报质量与公告风险
→ 技术位置与量价
→ 风险收益比
→ 短线策略判断
```

核心原则：

- 情绪环境不支持时，技术形态再强也要降低追高评级。
- 板块退潮时，后排股默认不追。
- 主线板块前排股优先观察分歧承接，后排补涨股优先等回踩或回避。
- 公司澄清热门概念、提示业务占比低、提示交易风险时，必须单独列为情绪风险。
- 财报明显恶化时，即使短线情绪强，也必须降低风险收益评级。
- 没有可靠数据时，不要为了完成模板而编造结论。

## 3. 数据源与可信度规则

### 3.1 数据源分级

| 等级 | 数据类型 | 推荐来源 | 用途 |
|---|---|---|---|
| A 级 | 公告、监管、交易公开信息、融资融券、龙虎榜、定期报告 | 上交所、深交所、北交所、巨潮资讯、上市公司公告 | 判断硬事件、监管风险、财报和资金公开信息 |
| B 级 | 实时 / 延迟行情、成交额、换手率、量比、板块涨幅、资金流 | 券商行情软件、东方财富、同花顺、雪球、Wind / Choice 等 | 判断短线强弱、板块热度、盘口和量价 |
| C 级 | 新闻、研报、自媒体、社区讨论、用户截图、其他 AI 总结 | 财经媒体、社交平台、用户提供内容 | 只能作为线索，不能作为核心结论依据 |

数据优先级：

- A 级数据与 B / C 级数据冲突时，优先采用 A 级。
- 行情数据没有时间戳时，不得当作实时行情。
- 第三方平台的“主力资金流”口径差异较大，只能作为辅助，不得作为单独买卖依据。
- 用户截图必须检查截图时间；无法确认时间时，只能作为历史线索。

### 3.2 每次分析必须先说明数据状态

每次输出前必须交代：

```text
数据时间：
行情来源：
公告核验：
资金数据来源：
是否为盘中数据：
是否可能延迟：
缺失数据：
不可判断项：
```

如果用户问“现在能不能追”“下午怎么看”，但没有当天行情或盘口数据，只能做框架分析，不能给具体追高 / 低吸判断。

### 3.3 禁止编造数据

不得编造以下数据：

- 现价、涨跌幅、成交额、换手率、量比。
- MA5、MA10、MA20、VWAP、分时均价线。
- 主力净流入 / 净流出。
- 龙虎榜席位、机构买卖、游资席位、量化席位。
- 融资余额、融资买入额、融券余额。
- PE / PB、市值、营收、净利润、扣非净利润、现金流。
- 公告内容、监管状态、减持计划、澄清公告。
- 板块涨幅、板块成交额、同板块个股涨跌。

无法查证时，用“暂未查到可靠数据”“该项需要以实时行情软件或公告为准”“数据不足，不能判断”表达。

## 4. 市场情绪分析

短线分析必须先判断当前市场是否适合做短线。

重点观察：

- 涨停家数、跌停家数。
- 连板高度和晋级率。
- 高位股是否出现亏钱效应。
- 主线题材是否持续扩散。
- 前排核心是否继续走强。
- 后排补涨是否掉队。
- 是否存在冰点修复、情绪分歧、高潮退潮。

输出分类：

| 情绪状态 | 含义 | 短线策略 |
|---|---|---|
| 情绪强 | 涨停较多、主线扩散、前排继续加强 | 可关注主线前排分歧机会，不盲追后排 |
| 情绪分歧 | 高位开始分化，部分前排承接仍在 | 降低仓位假设，优先看核心，不追杂毛 |
| 情绪退潮 | 高位亏钱效应扩散，跌停或大面增多 | 后排回避，强势股也只看不追 |
| 冰点修复 | 前期大跌后开始止跌回暖 | 可观察低位修复和新主线，避免追旧高位 |

判断模板：

```text
当前更像情绪分歧期，不是全面退潮，但高位股已经开始出现亏钱效应。因此短线只适合看主线前排的承接，后排补涨股不适合追高。
```

---

## 5. 板块强弱与热门程度

不能只看个股，必须看板块。

### 5.1 板块判断维度

- 所属板块是否为当前市场主线。
- 板块涨幅和成交额是否靠前。
- 板块内涨停数量和大涨数量。
- 龙头是否继续走强。
- 前排是否扩散到中军、补涨、低位分支。
- 板块是持续发酵，还是一日游。
- 个股上涨是板块共振，还是孤立异动。

### 5.2 板块分类

| 分类 | 特征 | 判断 |
|---|---|---|
| 主线板块 | 持续多日活跃，龙头强，扩散充分 | 优先关注前排和核心中军 |
| 支线轮动 | 有表现但持续性一般 | 只看低位或强承接，不追高 |
| 一日游题材 | 单日爆发，缺少持续龙头 | 谨慎，次日容易分化 |
| 退潮板块 | 龙头分歧，后排掉队，亏钱效应扩散 | 后排默认不追 |

判断模板：

```text
该股今天强，不是孤立上涨，背后有 XXX 板块联动。但板块内真正强的是前排 A / B，该股更偏后排补涨。若龙头明天分歧，它的回撤弹性可能更大。
```

---

## 6. 个股板块地位

短线优先看地位，而不是只看涨幅。

必须判断个股属于：

| 地位 | 特征 | 短线含义 |
|---|---|---|
| 龙头 | 最先启动、涨幅和辨识度最高、带动板块 | 可观察分歧承接，但高位不无脑追 |
| 前排 | 跟随龙头强势，流动性和辨识度较好 | 优先级较高，适合看回踩和承接 |
| 中军 | 市值较大、容量较好、趋势更稳 | 适合趋势观察，不一定适合连板追逐 |
| 后排 | 跟风上涨，辨识度较弱 | 板块分歧时优先回避 |
| 补涨 | 龙头涨高后资金挖掘低位 | 只适合早期，高潮后不追 |
| 纯情绪票 | 基本面和板块支撑弱，主要靠情绪 | 风险高，必须降低仓位假设和追高评级 |

输出必须给出：

```text
板块地位：龙头 / 前排 / 中军 / 后排 / 补涨 / 跟风 / 纯情绪票 / 数据不足
```

## 7. 参与者结构 / 筹码成分

短线要判断是谁在买、谁可能在卖，但不能编造具体占比。

### 7.1 可参考数据

- 龙虎榜：机构席位、游资席位、量化席位、沪股通 / 深股通。
- 融资融券：融资买入、融资余额变化。
- 成交额、换手率、量比。
- 分时走势：急拉急跌、反复脉冲、均价线承接。
- K 线特征：高位巨量、缩量回踩、放量滞涨。
- 股东户数、机构持仓、基金持仓，只能作为中期筹码背景。

### 7.2 参与结构分类

| 分类 | 常见特征 | 判断意义 |
|---|---|---|
| 游资主导 | 龙虎榜活跃席位、涨停封板、隔日波动大 | 强度高但兑现快 |
| 量化活跃 | 高频脉冲、反复拉扯、成交活跃但持续性弱 | 容易冲高回落 |
| 机构趋势 | 走势较稳、容量较大、趋势延续 | 更适合趋势观察 |
| 散户博弈 | 消息驱动、换手高、承接弱 | 不稳定，追高风险大 |
| 融资盘偏重 | 融资余额高或融资买入大 | 下跌时容易踩踏 |
| 数据不足 | 无龙虎榜 / 融资 / 成交结构可靠数据 | 不做结构判断 |

硬规则：

- 不得编造量化、游资、机构、散户的具体占比。
- 没有龙虎榜、融资融券、成交结构或可靠分时数据时，只能输出“数据不足”。
- 若资金流与价格行为冲突，优先看价格、成交额、换手率和板块联动。

判断模板：

```text
从成交和分时看，更像短线资金主导，而不是机构趋势买入。若龙虎榜没有机构持续买入配合，明天更关键的是承接，而不是今天涨幅本身。
```

## 8. 财报质量与基本面风险

短线不做长期估值模型，但必须判断财报是否会影响情绪。

重点检查：

- 营收同比、净利润同比、扣非净利润。
- 毛利率、现金流、负债率。
- 业绩预告、业绩快报、定期报告。
- 是否亏损、扭亏、预减、预增。
- 当前题材是否能被财报支持。
- 估值是否明显透支。

输出分类：

| 财报影响 | 含义 |
|---|---|
| 财报支撑 | 业绩改善或基本面能支撑题材 |
| 财报一般 | 没有明显加分，也没有明显拖累 |
| 财报拖累 | 业绩下滑、亏损、现金流或负债压力较大 |
| 业绩雷风险 | 存在预亏、暴雷、审计、退市或重大不确定性 |
| 数据不足 | 未查到可靠财报数据 |

规则：

- 财报明显恶化时，即使短线情绪强，也必须降低风险收益评级。
- 题材炒作与主营收入占比不匹配时，必须提示“题材与业绩支撑可能脱节”。

---

## 9. 公告、监管与澄清风险

如果存在以下事件，必须单独分析：

- 异常波动公告。
- 严重异常波动公告。
- 监管函、问询函、重点监控、出监管。
- 大股东减持、董监高减持。
- 澄清热门概念公告。
- 公司提示某业务收入占比很低。
- 公司提示股价涨幅与基本面不匹配。
- 复牌、摘帽、并购、重大合同、政策催化、价格上涨。

### 9.1 澄清公告的短线影响

澄清公告可能直接影响情绪，必须重点判断：

- 公司是否否认涉及热门概念。
- 公司是否承认涉及但收入占比很低。
- 公司是否提示尚未产生收入或存在不确定性。
- 公告发布前股价是否已经大涨。
- 公告后资金是否仍继续承接。

输出分类：

```text
公告情绪影响：正向 / 中性 / 负向 / 高风险 / 未核验
```

判断模板：

```text
公司已经澄清该热门概念收入占比较低，这会削弱题材想象空间。若明天还能放量站稳，说明资金仍按情绪博弈处理；若高开低走，则容易变成利好兑现或概念证伪后的分歧。
```

## 10. 技术面分析

技术面必须服务于短线风险收益比。

重点看：

- 近期走势脉络：从哪天启动，涨到哪里，中间是否洗盘，是否反包。
- 是否处于主升、加速、分歧、回调、反包、破位阶段。
- MA5、MA10、MA20 的支撑关系。
- 当前价格距离均线是否过远。
- 阶段高点 / 低点位置。
- 是否出现高位放量滞涨、长上影、冲高回落。
- 盘中是否站稳分时均价线 / VWAP。

常用判断：

| 形态 | 判断 |
|---|---|
| 沿 MA5 上行 | 短线强趋势 |
| 跌破 MA5 但守 MA10 | 强势转震荡 |
| 放量跌破 MA10 | 短线明显转弱 |
| 大跌后快速反包 | 有资金回补，需看次日承接 |
| 涨停后次日高开低走 | 获利盘兑现，追高风险增加 |
| 缩量回踩 MA5 / MA10 | 可能是良性洗盘 |
| 放量下跌且无法收回均线 | 分歧转弱或资金出逃 |

## 11. 观察位生成规则

每只股票尽量给出关键观察位，但必须基于可靠行情数据。

### 11.1 支撑位优先级

1. 盘中分时均价线 / VWAP 附近。
2. 当日低点。
3. 前一交易日收盘价。
4. MA5。
5. MA10。
6. 前平台突破位。
7. 涨停启动日实体区间。

### 11.2 压力位优先级

1. 当日前高。
2. 昨日高点。
3. 近期阶段高点。
4. 涨停开板位置。
5. 高位放量阴线高点。
6. 整数关口。

硬规则：

- 没有可靠 K 线和实时价格时，不输出精确价格。
- 数据不足时，只能输出“MA5 附近”“前高附近”“当日低点附近”等相对位置。
- 不得随口编造支撑位、压力位或止损价。

---

## 12. 量价与资金面分析

必须判断涨跌和成交是否匹配。

| 量价组合 | 含义 |
|---|---|
| 放量上涨 | 资金主动进攻，短线强度较高 |
| 放量涨停 | 情绪强，但次日看承接，不能直接认定继续涨 |
| 放量下跌 | 高位兑现或资金分歧，风险上升 |
| 缩量回调 | 若不破关键均线，可能是良性洗盘 |
| 缩量反弹 | 反弹力度不足，需补量确认 |
| 高位巨量横盘 | 筹码剧烈交换，可能是分歧，也可能是派发 |

分析语气要明确：

```text
今天不是普通小回调，而是高位放量下跌，说明抛压明显，不能简单理解为洗盘。若明天缩量止跌并重新站回 MA5，才说明承接仍在。
```

## 13. 操作判断标准

### 13.1 是否适合追

| 结论 | 适用条件 |
|---|---|
| 不适合追 | 高位放量分歧、涨停后高开低走、离均线过远、风险公告较多、板块退潮 |
| 可以观察 | 趋势未坏，但位置偏高，需要等承接确认 |
| 只适合低吸 | 板块逻辑仍在，但追高没有安全边际 |
| 可小仓试错 | 回踩关键位缩量企稳，止损位清楚，且板块情绪未退潮 |
| 只看不碰 | 估值过高、减持 / 监管 / 融资盘风险重、短线涨幅严重透支、概念被澄清 |

### 13.2 走强与转弱条件

每只股票必须给出：

- 走强条件：放量站回关键位、突破前高、回封涨停、板块前排继续加强。
- 转弱条件：放量跌破 MA5 / MA10、跌破前低、冲高回落、板块龙头分歧、公告证伪题材。

---

## 14. 三种分析模式

### 14.1 盘中模式

适合用户问“现在能不能追”“下午怎么看”。

输出重点：

```text
1. 当前是否能追：能 / 不能 / 只能观察
2. 盘中强弱：主动进攻 / 被动跟涨 / 冲高回落 / 承接较弱
3. 关键盘口：量能、分时均价线、回落幅度、是否破日内低点
4. 下午观察：站上哪里转强，跌破哪里回避
```

### 14.2 收盘后模式

适合用户问“明天怎么看”。

输出重点：

```text
1. 今日 K 线性质
2. 量价是否健康
3. 板块是否延续
```

## 17. 话术风格

要求：

- 先概括要点，再展开。
- 语言直接，不绕弯。
- 不阿谀，不制造确定性。
- 多用“如果……则……”的条件判断。
- 风险和机会都要讲，但短线优先讲风险收益比。
- 用户问“能不能追”时，必须给明确态度。

常用表达：

```text
这票不是不能看，而是现在追没有安全边际。
```

```text
板块还在，但它不是前排，追高性价比不如等分歧。
```

```text
公司已经澄清相关概念收入占比有限，这会削弱短线题材想象空间。
```

```text
明天关键不是高开多少，而是高开后能不能站住分时均价线。
```

```text
如果板块龙头分歧，而它又只是后排补涨，就不能按强趋势处理。
```

## 18. 禁止事项

- 禁止编造实时行情和资金数据。
- 禁止编造量化、游资、机构、散户的具体占比。
- 禁止将截图中的旧数据当作最新数据。
- 禁止只看涨跌幅，不看情绪、板块和风险。
- 禁止只讲机会，不讲下跌空间。
- 禁止用“必涨”“稳赚”“无脑上”等表达。
- 禁止给出违反合规和风险控制的确定性交易指令。
- 禁止在缺少可靠数据时输出精确支撑位、压力位或止损价。
- 禁止把第三方主力资金流当作确定事实。
